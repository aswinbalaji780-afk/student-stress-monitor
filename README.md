# 🧠 Student Stress Monitor

A smart web application that helps students monitor their daily stress levels using lifestyle data, mood, typing behavior, and a short daily journal. The application analyzes multiple factors and predicts whether the student's stress level is **Low**, **Moderate**, or **High**, while also providing personalized suggestions to improve mental well-being.

> 🚀 This is my **first Vibe Coding project**, where I built an AI-powered stress monitoring application focused on improving student mental health.

---

## 📌 Features

- 😴 Track daily sleep duration
- 😊 Record current mood
- 📖 Analyze daily journal/description
- ⌨️ Measure typing speed during text input
- 🧠 Predict stress level based on user inputs
- 💡 Provide personalized stress management tips
- 📱 Clean and responsive user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- AI-assisted Vibe Coding
- Browser APIs

---

## 📊 Inputs Collected

The application collects the following information:

- Sleep duration (hours)
- Current mood
- Daily activity description
- Typing speed (Words Per Minute)
- Typing behavior while writing

---

## 🧠 Stress Analysis

The system evaluates multiple factors such as:

- Sleep quality
- Emotional mood
- Daily journal sentiment
- Typing speed
- User behavior patterns

Based on these factors, it predicts one of the following stress levels:

- 🟢 Low Stress
- 🟡 Moderate Stress
- 🔴 High Stress

---

## 💡 Recommendations

After the analysis, the application provides personalized suggestions such as:

- Improve sleep schedule
- Take regular study breaks
- Practice breathing exercises
- Stay hydrated
- Exercise regularly
- Talk to friends or family
- Seek professional help if stress remains high
